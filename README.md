# ⚡ DEV.PERSONAL — Portafolio de Consultoría & Desarrollo Ágil

**DEV.PERSONAL** es una landing page de alto impacto diseñada para desarrolladores freelance y consultores técnicos. Con una estética inspirada en interfaces de comandos (CLI) y un enfoque en el rendimiento, este portafolio busca proyectar profesionalismo, agilidad y dominio técnico.

## ✨ Características Destacadas

* **⌨️ Command Palette (⌘ + K):** Navegación rápida mediante una paleta de comandos interactiva, permitiendo al usuario saltar entre secciones o ver proyectos sin usar el ratón.
* **🎨 Estética Cyber-Tech:** Diseño oscuro con acentos neón, efectos de ruido (grain), scanlines y orbes dinámicos de fondo para una experiencia inmersiva.
* **📂 Portfolio Modal System:** Detalle profundo de cada proyecto mediante un sistema de modales que permite explicar el Problema, la Solución y los Resultados.
* **📑 Capabilities Tabs:** Clasificación inteligente de servicios mediante pestañas filtrables (Automatización, Infraestructura, Game Dev, IA).
* **📱 Diseño Ultra-Responsivo:** Adaptación fluida desde monitores 4K hasta dispositivos móviles, manteniendo la legibilidad y el estilo.
* **🚀 Vibe Coding Workflow:** Integración visual de procesos acelerados por IA en la sección de "Método".

## 🛠️ Stack Técnico

* **HTML5 Semántico:** Estructura limpia y accesible.
* **CSS3 Avanzado:** * Variables (Custom Properties) para gestión de temas.
    * Animaciones clave (@keyframes) y efectos de desenfoque (backdrop-filter).
    * Grid y Flexbox para layouts complejos.
* **JavaScript (Vanilla):**
    * Lógica de navegación y filtrado sin dependencias externas.
    * Intersection Observer API para revelado de contenido al hacer scroll.
    * Gestión de eventos de teclado para accesos rápidos.

## 📦 Instalación y Uso

1.  **Clona el repositorio:**
    ```bash
    git clone [https://github.com/tu-usuario/dev-personal-portfolio.git](https://github.com/tu-usuario/dev-personal-portfolio.git)
    ```
2.  **Abre el archivo principal:**
    Simplemente abre `index.html` en cualquier navegador moderno. No requiere servidores ni procesos de compilación.

## ⚙️ Personalización

Para adaptar este portafolio a tu propia marca personal:

1.  **Colores:** Edita las variables `:root` en el `<style>` para cambiar los tonos de acento (`--accent`, `--accent2`).
2.  **Proyectos:** Modifica el objeto `PROJECTS` en el bloque de `<script>` al final del archivo para actualizar tus casos de éxito.
3.  **Comandos:** Añade o quita funciones en el array `CMDS` para personalizar la paleta de búsqueda.

## 🔒 Privacidad

Este sitio es estático y no recolecta datos del usuario. Las interacciones se procesan localmente en el navegador.
